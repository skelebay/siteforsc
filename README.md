# Система обмена заданиями для класса

Локальный веб-сайт для обмена заданиями между учителем и учениками в компьютерном классе.

## Возможности

- **Для учителя:**
  - Загрузка заданий для всех учеников
  - Просмотр и скачивание работ учеников
  - Удаление загруженных заданий

- **Для учеников:**
  - Просмотр и скачивание заданий от учителя
  - Загрузка выполненных заданий
  - Просмотр своих загруженных работ

## Установка на сервер (Debian)

### 1. Установите Python и pip (если еще не установлены)

```bash
sudo apt update
sudo apt install python3 python3-pip -y
```

### 2. Установите зависимости

```bash
pip3 install -r requirements.txt
```

### 3. Запустите сервер

```bash
python3 app.py
```

Сервер будет доступен по адресу `http://IP_адрес_сервера:5000`

Чтобы узнать IP-адрес сервера:
```bash
hostname -I
```

## Настройка пользователей

По умолчанию созданы следующие учетные записи:

- **Учитель:** `teacher` / `teacher123`
- **Ученик 1:** `student1` / `student123`
- **Ученик 2:** `student2` / `student123`

Чтобы добавить больше пользователей, откройте файл `app.py` и найдите словарь `USERS`. Добавьте новых пользователей в формате:

```python
'username': {'password': 'password123', 'role': 'student'},
```

## Доступ с компьютеров учеников

1. Убедитесь, что все компьютеры подключены к одной локальной сети
2. Откройте браузер на компьютере ученика
3. Введите адрес: `http://IP_адрес_сервера:5000`
4. Войдите используя свои учетные данные

## Запуск в фоновом режиме (для постоянной работы)

Чтобы сервер работал постоянно, даже после закрытия терминала:

```bash
nohup python3 app.py > server.log 2>&1 &
```

Для остановки сервера:
```bash
pkill -f "python3 app.py"
```

## Безопасность

⚠️ **Важно:** Это приложение предназначено для использования в локальной сети класса. Для продакшн-использования необходимо:

1. Изменить `secret_key` в `app.py` на случайную строку
2. Использовать настоящую базу данных вместо словаря пользователей
3. Добавить шифрование паролей (хеширование)
4. Настроить HTTPS для защиты данных

## Решение проблем

**Проблема:** Не могу подключиться с компьютера ученика
- Проверьте, что сервер запущен
- Убедитесь, что компьютеры в одной сети
- Проверьте настройки файрвола на сервере:
  ```bash
  sudo ufw allow 5000
  ```

**Проблема:** Ошибка при установке зависимостей
- Используйте `pip3` вместо `pip`
- Попробуйте: `python3 -m pip install -r requirements.txt`
